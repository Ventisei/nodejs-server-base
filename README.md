# nodejs-server-base
NodeJS Server Base Structure (ready to use)

## Instructions
### 1 Install dependencies
```
npm i
```

### 2 Set enviroment variables
Rename ```.env.example``` to ```.env``` and set variables.

### 3.1 Start server
```
npm start
```

### 3.2 Start server in development mode (restart on each file change)
```
npm run dev
```
